# 《回声炼金师》UE4 游戏项目

这是一个基于 Unreal Engine 4/5 的游戏项目，旨在探索和实践人机协同开发的工作流。

## 核心规范

本项目严格遵循 [gdszyy/ue4-human-ai-collab](https://github.com/gdszyy/ue4-human-ai-collab) 仓库中定义的**人机协同开发规范**和**Linear工作流规范**。

- **任务管理**: Linear (VOI-5)
- **代码仓库**: GitHub (本仓库)
- **设计文档**: Notion (GDD)

请参考以下文档了解详细信息：
- [人机协同开发规范](https://github.com/gdszyy/ue4-human-ai-collab/blob/main/docs/collaboration_specification.md)
- [Linear工作流规范](https://github.com/gdszyy/ue4-human-ai-collab/blob/main/docs/workflow_specification.md)

## Git LFS 配置

本项目已启用 Git LFS (Large File Storage) 来管理 UE4 的大文件资产。

**已跟踪的文件类型**:
- `*.uasset`
- `*.umap`
- `*.fbx`
- `*.png`
- `*.jpg`
- `*.wav`
- `*.mp3`

在克隆或拉取项目前，请确保您已安装并初始化 Git LFS：
```bash
git lfs install
```
